## .dotfiles

macOS dotfiles, application settings, and user defaults


### Bootstrapping

1. `clone git@github.com:WilHall/.dotfiles.git ~/.dotfiles/`
1. `cd ~/.dotfiles`
1. `./bootstrap <DOTFILES_GITHUB_PERSONAL_ACCESS_TOKEN>`
